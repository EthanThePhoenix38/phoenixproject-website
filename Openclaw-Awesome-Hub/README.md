## 🏗️ Comment ça Fonctionne / How It Works

```
📡 GitHub API (every hour)
    ↓
🔍 Scan trending + keywords (openclaw, clawbot, moltbot, open-claw...)
    ↓
🧹 Filter: no forks, sort by stars, deduplicate
    ↓
🤖 Auto-translate EN → FR via script
    ↓
📄 Update README + generate HTML pages
    ↓
🌿 Branch "Links" ← 5 new links per auto-commit + silent PR
    ↓
🌐 Deploy to GitHub Pages
```
