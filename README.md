## 🇬🇧 English Version

Modern multilingual website for The Phoenix Agency - AI, NoCode, and Automation services.

### 🌐 View the Site

**[Access the website](https://ethanthePhoenix38.github.io/phoenixproject-website/)**

### ✨ Features

#### Internationalization
- 🌍 FR/EN localization system
- 🔄 Automatic language detection via IP
- 🔘 Language switcher button (iOS style)

#### Themes & Templates
- 🌓 Light/Dark mode with icon toggle (🌙/☀️)
- 🎨 Optimized WCAG AA dark mode colors
- 🎭 4 visual templates:
  - **Modern** - Contemporary design with gradients
  - **Professional** - Elegant corporate style
  - **Fluid** - Dynamic animations
  - **Minimalist** - Clean black & white

#### User Experience
- 📱 Responsive design (Mobile First)
- 🧭 Dynamic navigation menu with active state
- 📑 Collapsible tab navigation (slide to hide/show)
- 🎯 Persistent menu state (saved in localStorage)
- ♿ WCAG AA compliant

#### Compliance & Data
- 🍪 GDPR cookie banner with consent
- 📊 Visit statistics by country
- 💾 User preferences storage

#### Interactive Demos
- 📊 **Statistics Dashboard** - Real-time visitor analytics

#### SEO & Security
- 🔍 Complete SEO meta tags (Open Graph, Twitter Cards)
- 🗺️ XML sitemap for search engines
- 🤖 robots.txt with AI bot protection
- 🔒 Commercial proprietary license
- 🛡️ Anti-scraping measures

### 📱 Supported Devices

- iPhone Classic (375px - 414px)
- iPhone 16 Pro Max (430px)
- iPad (768px - 1024px)
- 22-inch PC (1920px)
- 34-inch Ultra-Wide Monitor (3440px)

### 🛠️ Tech Stack

**Core Technologies:**
- HTML5 semantic
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript ES6+
- ipapi.co API for geolocation

**Featured Technologies (33+):**
Python, JavaScript, TypeScript, React, Vue.js, Node.js, FastAPI, AI Agents, PyTorch, OpenAI GPT-4, Claude AI, Cursor AI, VS Code, MCP, Autogravity, LangChain, n8n, Make, Airtable, PostgreSQL, MongoDB, Redis, Linux, GitHub Actions, CI/CD, REST API, GraphQL, WebSockets, Git, Agile/Scrum, CBAP

### 📦 Project Structure

```
├── index.html           # Main HTML structure
├── styles.css           # Themes and responsive design
├── templates.css        # 4 template styles
├── theme.js            # Interactive features
├── locales.js          # Translation system
├── statistiques.html   # Visitor statistics dashboard
├── robots.txt          # SEO and bot control
└── sitemap.xml         # XML sitemap for search engines
```
